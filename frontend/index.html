OBJETIVO:
Ser o primeiro a baixar todas as suas peças.

Cada jogador recebe 7 peças aleatórias.

O jogador que tiver a peça mais alta é o primeiro a jogar.

Em cada turno, o jogador correspondente deve colocar uma das suas peças numa das pontas do dominó no campo de jogo desde que, pelo menos uma das pontas tenha o mesmo valor que o valor da ponta escolhida do dominó.

Caso não consiga jogar uma peça válida, o jogador deve tirar novas peças do baralho até obter uma que encaixe no tabuleiro.

Se o baralho estiver vazio, o jogador tem de passar o turno.

Se o adversário também não conseguir jogar então é considerado empate.
